GCN for CECS 551

Implements GCN for CORA Citeseer and Pubmed datasets in an effort to reproduce results in the paper Semi-Supervised Classification with Graph Neural Networks by Kipf and Welling.

Implementation is directly inspired by Kipf's pytorch implemetnation.

Usage

The following commands assume you have Anaconda (or miniconda) installed on your system.

Create the environment

We have provided an environment.yml file with all required dependencies defined. To create the environment, run:

bash conda update --all # Make sure anaconda is up-to-date conda env create -f environment.yml # this should be run only once conda activate gcn551 # this must be done in every new shell

Running

See main.py for an example of usage. You can run this file on a dataset to test with the default hyperparameters:

bash python main.py /path/to/data <cora|citeseer|pubmed>
